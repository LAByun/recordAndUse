# 更新日志与使用手册

## 如何使用
在账户中优先输入个人的姓名，方便反馈记录
在日志中输入相关问题观察内容
聊天下面是AI参考的知识库可以用于查看
右下角为反馈按钮可以点击反馈个人问题

## 当前功能（2025-4-15）

- 流式回复回答用户的问题
- 将自己参考的数据库反馈出来
- 展示思考过程
- 加入了清除对话机制
- 记忆每一次对话，关联上下文，在用户未清空对话前，对话具有连贯性
